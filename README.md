## Development

###### Installation
```sh
$ git clone https://github.com/arnaudjuracek/www-bam
$ cd www-bam
$ npm install
```

###### Usage
```sh
$ npm run start # start kirby-webpack devserver
$ npm run build # build and bundle src/ into www/assets/
```
